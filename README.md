# The Web3 Platform

Web3 API's turned into React Hooks. #useTheWeb3Platform

## Install  nvm v8.0.0

  curl https://raw.githubusercontent.com/creationix/nvm/v0.16.1/install.sh | sh
  
  source ~/.profile
  
  nvm install v8.0.0
  
  nvm use v8.0.0
  
## Install web3 libs

  npm install
